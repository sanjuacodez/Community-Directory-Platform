# Community Directory Platform

## Vision

Build a web-based community directory platform that allows communities to manage families, members, relationships, announcements, events, businesses, jobs, and obituary records.

The system should be designed as a reusable SaaS-ready platform capable of supporting multiple communities in the future.

## Primary Objectives

* Centralized member directory
* Family relationship mapping
* Community communication
* Business and job discovery
* Historical family records

## Non Goals (V1)

* Mobile App
* Matrimony
* QR Family Cards
* Payment Gateway
* Chat System

## Target Users

1. Super Admin
2. Family Admin
3. Community Members
4. Public Visitors (optional)

## Estimated Scale

Initial:

* 1 Community
* 10-50 Families
* 100-1000 Members

Future:

* Multi Community SaaS
* 100+ Communities
* 100,000+ Members
